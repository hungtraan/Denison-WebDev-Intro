CSx 001: Intro to Web Development

Building (& Deploy) Your First Web Application

You are a tech enthusiast/ a Computer Science student? You feel the need to learn more about the practical application of what youâ€™ve learnt in the formal classroom? You want to be able to build any web application that you can think of? This class will provide you with the necessary knowledge to start building your first website/web application/ personal project.

Topics: This class focuses on the practical application of computer science in the Internet age. Students will learn about the following topics by examining their roles in popular websites, and eventually apply this knowledge to small projects and a final project

Everyday elements of web: the web application, front-end, back-end, server-side, client-side, database Architectural elements of the web: web server, web address, network/system administration From static to interactive + responsive: HTML5/CSS3, Javascript and jQuery, Bootstrap From static to dynamic: Database (relational and non-relational); Web development language & frameworks: Python, PHP, Ruby on Rails; REST, AJAX Development practices: command line, collaboration & version control (Git), tools Deployment/System Administration: Heroku, Amazon Web Services Stand on shoulders of the giants: Leveraging the world of libraries and APIs (Introduction of) New technologies: HTTP/2, AngularJS, React, Cassandra, Polymer The class will focus on breadth. In this class, learning is by example and by doing, as all activities are aimed to be as interactive as possible. Prerequisite: Fundamental understanding of computer science: loop, conditionals, functions. The level of the class will be adjusted according to overall level of students. Final Project: In this project students will employ the skills they have acquired earlier to design and implement a web development project of their interest.
